Public Methods

setActiveSlide

since v0.0.1 
        
          <script>
            (async () => {
              await customElements.whenDefined('safety-slider');
              const safetySliderElement = document.querySelector('safety-slider');
              await safetySliderElement.setActiveSlide(1);
            })();
          </script>

Integrations

Website integration

safety-slider has files specifically generated for website usage in the www/build directory. Place those files together where you would like them to be, then add the following script tags to the end of the HTML body tag. 

        
          <script type="module" src="/path-to-files/safety-slider.esm.js"></script>
          <script nomodule src="/path-to-files/safety-slider.js"></script>

Styling

The styling of safety-slider is the bare minimum. This is intentional as many will prefer to add their own styling anyway.

Note that hiding or altering the content of buttons can be achieved by the API properties defined above.

The following CSS variables are available to change from their default values. 

        
          // Scoped to <safety-slider>
          --safety-slider-bottom-padding: 2rem;
          --safety-slider-x-padding: 3rem;

          // Scoped to <safety-slider-arrows>
          --safety-slider-arrow-border-width: 1px;
          --safety-slider-arrow-border-color: black;
          --safety-slider-arrow-background-color: white;
          --safety-slider-arrow-padding: 0;
          --safety-slider-arrow-color: black;
          --safety-slider-arrow-width: 2rem;
          --safety-slider-arrow-height: var(--safety-slider-arrow-width);
          --safety-slider-arrow-disabled-background-color: gainsboro;

          // Scoped to <safety-slider-dots>
          --safety-slider-dots-container-height: calc(var(--safety-slider-bottom-padding) / 2);
          --safety-slider-dots-container-position-bottom: calc(var(--safety-slider-bottom-padding) / 8);
          --safety-slider-dot-background-color: black;
          --safety-slider-dot-width: .5rem;
          --safety-slider-dot-height: var(--safety-slider-dot-width);
          --safety-slider-dot-x-margin: calc(var(--safety-slider-dot-width) / 2);
          --safety-slider-dot-disabled-background-color: gainsboro;
